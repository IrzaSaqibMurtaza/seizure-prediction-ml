# Seizure Prediction using Logistic Regression

This project investigates the effect of preprocessing pipelines,
regularization strategies, and class imbalance handling on seizure prediction.

## Datasets
- Kaggle Epileptic Seizure Recognition
- UCI BEED Dataset
- Bonn EEG Dataset

## Techniques
- Logistic Regression
- L1 / L2 / Elastic Net
- PCA
- Feature Selection
- SMOTE
- Class Weighting

## Tools
- Python
- Google Colab
- Scikit-learn
